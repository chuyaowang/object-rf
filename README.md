**object-rf** is a napari plugin designed for object-level classification using Random Forest. It is intended to be used as a downstream processing tool for pixel-level predictions (such as those generated by [napari-rf](https://github.com/chuyaowang/napari-rf)).

## Overview

The plugin provides a workflow to:
1.  **Segment Objects**: Convert pixel-level probability maps into discrete object labels via thresholding and connected component analysis.
2.  **Extract Features**: Calculate object-level features (e.g., geometry, intensity, texture) for each identified object using `scikit-image.measure.regionprops`.
3.  **Classify Objects**: Train and apply a Random Forest classifier to categorize objects (e.g., identifying different cell types or states).

## Installation

You can install `object-rf` via [pip]:

```bash
pip install object-rf
```

## Workflow

- **Input**: Pixel-level probability layers or segmentation masks.
- **Processing**:
    - Thresholding and labeling of probability maps.
    - Automatic feature extraction (area, eccentricity, mean intensity, etc.).
    - Interactive training on specific objects.
- **Output**: Labeled objects categorized by class.
